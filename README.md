# 🏢 RD Service - Site Institucional

Um site corporativo moderno e responsivo desenvolvido para a empresa de serviços RD Service. O foco deste projeto foi criar uma presença digital profissional, garantindo uma interface rápida para os clientes e um sistema de contato direto totalmente funcional.

🔗 **Acesso ao projeto em produção:** [rdservice.tec.br](https://rdservice.tec.br)

## 💻 Tecnologias e Ferramentas Utilizadas

* **Front-end:** React.js, JavaScript (ES6+), HTML5 e CSS3
* **Armazenamento e Dados:** Firebase (Integração de formulário e gerenciamento de mensagens)
* **Versionamento e Infraestrutura:** Git, GitHub e configuração de domínio customizado (DNS)

## ✨ Principais Funcionalidades

* **Design Responsivo (Mobile-First):** Layout 100% adaptável para proporcionar a melhor navegação do usuário em celulares, tablets e computadores.
* **Formulário de Contato Dinâmico:** Os clientes da empresa podem enviar mensagens diretamente pela página de contato.
* **Integração com Firebase:** Todos os dados inseridos no formulário são enviados, salvos e gerenciados de forma segura no banco de dados do Firebase, sem a necessidade de um servidor próprio complexo.
